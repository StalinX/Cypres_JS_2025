# Cypres_JS_2025

Cypress con JavaScript Mayo 2025

## Descripción

Este proyecto contiene pruebas automatizadas utilizando [Cypress](https://www.cypress.io/) con JavaScript.

## Instalación

1. Clona el repositorio:
   ```sh
   git clone <url-del-repositorio>
   cd Cypres_JS_2025
   ```

2. Instala las dependencias:
   ```sh
   npm install
   ```

## Ejecución de pruebas

Para ejecutar las pruebas en modo interactivo:
```sh
npx cypress open
```

Para ejecutar las pruebas en modo headless:
```sh
npx cypress run
```

## Estructura del proyecto

- `cypress/` - Contiene los tests, fixtures y soporte.
- `node_modules/` - Dependencias del proyecto.
- `package.json` - Configuración y scripts de npm.
- `.gitignore` - Archivos y carpetas ignorados por git.
